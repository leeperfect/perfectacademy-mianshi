# 文件夹使用规则

## research 文件夹
- **用途**：仅用于存放教研完成的合格成品文件
- **文件要求**：
  - 文件内容为最终版、可直接使用的教研成果
  - 不得存放素材类文件或未完成的工作文件

## materials 文件夹
- **用途**：用于存放各类素材文件，包括参考资料、原始数据、未完成的工作文件等
- **子文件夹**：
  - `interview_confidence_training_materials`：存放面试自信训练相关的素材文件

## 命名规范
- **文件命名**：`[研究类型]_[具体内容]_[版本号]_[日期].md`
  - 示例：`confidence_training_面试气场提升方案_v1.0_20260407.md`
- 所有文件和文件夹命名统一使用英文小写字母及下划线 (`snake_case`)
- 确保文件名清晰反映文档内容
- 避免使用中文或特殊字符，确保跨平台兼容性

## 工作流程
1. 素材文件先存放在 `materials` 文件夹中进行整理和处理
2. 教研完成的成品文件移至 `research` 文件夹中
3. 定期清理 `materials` 文件夹，移除不需要的文件
4. 确保 `research` 文件夹中的文件都是最终版、可直接使用的成果
